# ESLint Config Sodaru

ESLint configurations From Sodaru

> Provides common eslint configurations for Sodaru projects

## Configs

- default - for typescript projects
  ```YAML
  extends:
    - sodaru
  ```
- react
  ```YAML
  extends:
    - sodaru/react
  ```
